Theme Modes

Dark and light mode using CSS custom properties and the prefers-color-scheme media query. Click the toggle in the nav to cycle between System, Light, and Dark.

How it works

Layer 1 — CSS custom properties define light-mode tokens on :root.

Layer 2 — A prefers-color-scheme: dark media query overrides those tokens automatically when the OS is in dark mode.

Layer 3 — A data-theme attribute on <html> lets the user override the OS preference. The choice is saved to localStorage.

Code block

:root {
  --bg: #ffffff;
  --fg: #1a1a2e;
}

@media (prefers-color-scheme: dark) {
  :root {
    --bg: #0f172a;
    --fg: #e2e8f0;
  }
}

No JavaScript required for the baseline — the browser handles it. The toggle adds a small script for explicit user control.
