# Kanban

A full-featured task board with auth, sessions, CSRF protection, htmx updates,
and SSE-driven activity. It is the most complete multi-user example in the
examples tree and a strong reference for app-shell style Chirp apps.

## Run

```bash
PYTHONPATH=src python examples/standalone/kanban/app.py
```

## Test

```bash
pytest examples/standalone/kanban/
```
