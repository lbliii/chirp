# Contacts

The canonical htmx CRUD example for Chirp. It demonstrates add, inline edit,
delete, search, `ValidationError`, out-of-band updates, and HX response headers
for toast-style UX without client-side application code.

## Run

```bash
PYTHONPATH=src python examples/standalone/contacts/app.py
```

## Test

```bash
pytest examples/standalone/contacts/
```
