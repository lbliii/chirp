# Dashboard

A live dashboard example that combines Chirp, Kida, and streaming UI patterns.
It renders a weather-style dashboard with live sensor data, cached fragments,
and server-driven updates so the page feels real-time without a frontend
framework.

## Run

```bash
PYTHONPATH=src python examples/standalone/dashboard/app.py
```

## Test

```bash
pytest examples/standalone/dashboard/
```
